Our First Android App

In this lesson, we will create our first Android app. We'll start by setting up a basic project structure and creating a simple user interface.

Android Studio Project Structure

Android Studio organizes projects into various directories and files. Let's take a look at the basic structure of an Android project:


MyFirstApp/
├── app/
│   ├── src/
│   │   ├── main/
│   │   │   ├── java/
│   │   │   │   └── com.example.myfirstapp/
│   │   │   │       └── MainActivity.java
│   │   │   ├── res/
│   │   │   │   ├── layout/
│   │   │   │   │   └── activity_main.xml
│   │   │   │   └── values/
│   │   │   │       └── strings.xml
│   │   │   └── AndroidManifest.xml
│   └── build.gradle
└── build.gradle
Creating User Interface

We'll start by creating a simple user interface using XML in Android Studio's layout editor.(activity_main.xml)


<?xml version="1.0" encoding="utf-8"?>
<androidx.constraintlayout.widget.ConstraintLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    tools:context=".MainActivity">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello World!"
        app:layout_constraintBottom_toBottomOf="parent"
        app:layout_constraintLeft_toLeftOf="parent"
        app:layout_constraintRight_toRightOf="parent"
        app:layout_constraintTop_toTopOf="parent" />

</androidx.constraintlayout.widget.ConstraintLayout>

Explanation: In this lesson, we learned how to set up our first Android app project in Android Studio and created a simple user interface using XML in the layout editor. We'll continue building upon this foundation in the upcoming lessons.

What is Java Programming?

Java is a high-level, object-oriented programming language developed by Sun Microsystems (now owned by Oracle Corporation) in the mid-1990s. It was designed to be platform-independent, meaning that Java programs can run on any device that has a Java Virtual Machine (JVM) installed.

Features of Java

Java is known for its simplicity, platform independence, security, and robustness. Some key features of Java include:

  • Object-oriented: Java follows the object-oriented programming paradigm, making it easy to organize and structure code.
  • Platform independence: Java programs can run on any device that has a JVM, regardless of the underlying hardware and operating system.
  • Automatic memory management: Java provides automatic garbage collection, relieving developers from managing memory allocation and deallocation.
  • Strongly typed: Java enforces strict type checking, reducing the likelihood of runtime errors.
  • Multi-threaded: Java supports multi-threading, allowing concurrent execution of multiple tasks within a single program.
  • Rich standard library: Java comes with a comprehensive standard library that provides ready-to-use classes and functions for common programming tasks.
Example of Java Code

Here's a simple "Hello, World!" program written in Java:


public class HelloWorld {
   public static void main(String[] args) {
       System.out.println("Hello, World!");
   }
}

Explanation: Java is a versatile and widely-used programming language known for its simplicity, platform independence, and robustness. It offers a rich set of features and a comprehensive standard library, making it suitable for a wide range of application development tasks.

Designing with XML

XML (eXtensible Markup Language) is a markup language that is commonly used for structuring and storing data in a human-readable format. In Android app development, XML is frequently used to define the layout and appearance of user interfaces.

XML Layouts in Android

In Android development, XML files are used to define the layout and appearance of user interfaces. These XML files are stored in the res/layout directory of the Android project. Let's take a look at an example of an XML layout file:


<?xml version="1.0" encoding="utf-8"?>
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
   android:layout_width="match_parent"
   android:layout_height="match_parent">

   <TextView
       android:id="@+id/textView"
       android:layout_width="wrap_content"
       android:layout_height="wrap_content"
       android:text="Hello, World!"
       android:layout_centerInParent="true" />

</RelativeLayout>

Explanation: XML is commonly used in Android app development to define the layout and appearance of user interfaces. Layout XML files are stored in the res/layout directory of the Android project and contain hierarchical descriptions of the UI elements.
