Video Uploading System

Here users to upload videos with their usernames. 


import os
import uuid
from flask import Flask, request, redirect, url_for, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/upload_video', methods=['POST'])
def upload_video():
    if request.method == 'POST':
        file = request.files['video']
        username = request.form['username']
        if file and username:
            folder_name = os.path.join('/path/to/videos_folder', username)
            os.makedirs(folder_name, exist_ok=True)
            filename = str(uuid.uuid4()) + '.mp4'
            file.save(os.path.join(folder_name, filename))
            return 'Video uploaded successfully.'
    return redirect(url_for('index'))

if __name__ == '__main__':
    app.run(debug=True)

Explanation: This Flask application provides a route '/' to render the index.html template and a route '/upload_video' to handle video uploads. When a video is uploaded, it is saved to a folder named after the username provided by the user. Each video file is assigned a unique filename generated using UUID.
