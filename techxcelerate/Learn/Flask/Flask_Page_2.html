In this example, we'll demonstrate how to pass variables from Flask to HTML templates.


from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    name = "John"
    age = 30
    return render_template('index.html', name=name, age=age)

if __name__ == '__main__':

Above is the Python code where we define a Flask route '/' and pass variables 'name' and 'age' to the 'index.html' template.


<!DOCTYPE html>

The above HTML code demonstrates how we access the variables 'name' and 'age' passed from Flask using Jinja2 templating syntax.


**Passing Data to HTML using Fetch JS Method**

In this example, we'll demonstrate how to use the Fetch API in JavaScript to retrieve data from a Flask server and display it on an HTML page.


from flask import Flask, render_template, request, jsonify

app = Flask(__name__)

@app.route('/')
def index():
   return render_template('index.html')

@app.route('/name')
def name():
   name = "John"
   return name

@app.route('/age')
def age():
   age = 37
   return age

if __name__ == '__main__':
   app.run(debug=True)

Above is the Flask server code that defines routes for retrieving the name and age.


<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Fetch Data</title>
</head>
<body>
   <div>
       <p id="name">Name: </p>
       <p id="age">Age: </p>
   </div>

   <script>
       fetch('/name')
           .then(response => response.text())
           .then(data => {
               document.getElementById('name').innerText += data;
           });

       fetch('/age')
           .then(response => response.text())
           .then(data => {
               document.getElementById('age').innerText += data;
           });
   </script>
</body>
</html>

The above HTML code demonstrates how to use the Fetch API to retrieve data from Flask routes '/name' and '/age', and then dynamically update the content of ` <p>` elements with the received data.

**Parsing Arrays and Lists with Fetch Method**

In this example, we'll demonstrate how to parse arrays and lists returned from a Flask server using the Fetch API.


from flask import Flask, jsonify

app = Flask(__name__)

@app.route('/')
def index():
   return render_template('index.html')

@app.route('/users')
def users():
   users = ["Alice", "Bob", "Charlie"]
   return jsonify(users=users)

if __name__ == '__main__':
   app.run(debug=True)

Above is the Flask server code that defines a route '/users' and returns an array of user names.


<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>User List</title>
</head>
<body>
   <ul id="userList"></ul>

   <script>
       fetch('/users')
           .then(response => response.json())
           .then(data => {
               const userList = document.getElementById('userList');
               data.users.forEach(user => {
                   const listItem = document.createElement('li');
                   listItem.textContent = user;
                   userList.appendChild(listItem);
               });
           });
   </script>
</body>
</html>

The above HTML code demonstrates how to use the Fetch API to retrieve an array of user names from the Flask route '/users' and dynamically create a list of users on the webpage.


Form Handling

In this example, we'll demonstrate how to handle form data in Flask and send it using the Fetch API.


<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Form Handling</title>
</head>
<body>
   <form id="myForm">
       <label for="name">Name:</label>
       <input type="text" id="name" name="name"><br>

       <label for="age">Age:</label>
       <input type="text" id="age" name="age"><br>

       <button type="submit">Submit</button>
   </form>

   <script>
       document.getElementById('myForm').addEventListener('submit', function(event) {
           event.preventDefault();
           
           const formData = new FormData(this);
           
           fetch('/submit-form', {
               method: 'POST',
               body: formData
           })
           .then(response => response.json())
           .then(data => {
               console.log(data);
           })
           .catch(error => {
               console.error('Error:', error);
           });
       });
   </script>
</body>
</html>

The above HTML code demonstrates a simple form with input fields for name and age. When the form is submitted, the data is sent to the '/submit-form' route in Flask using the Fetch API.


from flask import Flask, render_template, request, jsonify

app = Flask(__name__)

@app.route('/')

   return render_template('index.html')

   name = request.form['name']

   app.run(debug=True)

Above is the Flask server code that defines a route '/submit-form' to handle form submissions. It retrieves the name and age from the form data and returns a JSON response containing the submitted data.

**Parsing Images and Saving Them**

In this example, we'll show how to upload images using a form, parse them in Flask, and save them to the parent directory.


from flask import Flask, render_template, request
import os

app = Flask(__name__)

@app.route('/')
def index():
   return render_template('index.html')

@app.route('/upload', methods=['POST'])
def upload():
   if request.method == 'POST':
       file = request.files['file']
       if file:
           filename = file.filename
           file.save(os.path.join(os.getcwd(), filename))
           return 'File Uploaded Successfully'

if __name__ == '__main__':
   app.run(debug=True)

Above is the Flask server code that defines a route '/upload' to handle file uploads. It saves the uploaded file to the parent directory of the Flask app.


<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Image Upload</title>
</head>
<body>
   <form action="/upload" method="post" enctype="multipart/form-data">
       <input type="file" name="file" accept="image/*">
       <input type="submit" value="Upload">
   </form>
</body>
</html>

Here is the code with Fetch API

**Math Calculation**

To calculate the current flowing, by Flask and display the calculation result in HTML, follow these steps:


from flask import Flask, render_template, request, jsonify

app = Flask(__name__)

@app.route('/')
def index():
  return render_template('index.html')

@app.route('/calculate', methods=['POST'])
def calculate():
  if request.method == 'POST':
      voltage = float(request.form['voltage'])
      resistance = float(request.form['resistance'])
      current = voltage / resistance
      return jsonify({'current': current})

if __name__ == '__main__':
  app.run(debug=True)

Explanation:

  • Defines a Flask application.
  • Defines the route for the homepage ("/") and renders the "index.html" template.
  • Defines the "/calculate" route to handle POST requests for calculating the current.
  • Extracts the voltage and resistance values from the form data.
  • Calculates the current using Ohm's Law (current = voltage / resistance).
  • Returns the current value as JSON data.

<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <label for="voltage">Voltage (V):</label>

  • Creates a new FormData object to store the form data.
  • Sends a POST request to the "/calculate" endpoint with the form data.
  • Receives the current calculation result from the Flask server as JSON data.
  • Displays the current value in the resultDiv element.

** Conditional Statements**

This section demonstrates how to use conditional statements in Flask routes and handle them in HTML using the Fetch API along with a form to input age. 


from flask import Flask, render_template, request

app = Flask(__name__)

# Flask route with conditional statement
@app.route('/check_age', methods=['GET', 'POST'])
def check_age():
   if request.method == 'POST':
       age = int(request.form['age'])
       if age >= 18:
           return 'You are an adult.'
       else:
           return 'You are a minor.'
   return render_template('index.html')

if __name__ == '__main__':
   app.run(debug=True)

Explanation: This Python code defines a Flask route '/check_age' that accepts both GET and POST requests. When a POST request is received, it extracts the age from the form data and performs a conditional check. If the age is greater than or equal to 18, it returns 'You are an adult.' Otherwise, it returns 'You are a minor.' The route renders the 'index.html' template for GET requests. 


<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   <title>Conditional Statements</title>
</head>
<body>
   <form action="/check_age" method="post">
       <label for="age">Enter your age:</label>
       <input type="number" id="age" name="age" required>
       <button type="submit">Submit</button>
   </form>
   <div id="age_result"></div>

   <!-- JavaScript to fetch age data from Flask -->
   <script>
       document.querySelector('form').addEventListener('submit', function(event) {
           event.preventDefault();
           fetch('/check_age', {
               method: 'POST',
               body: new FormData(document.querySelector('form'))
           })
           .then(response => response.text())
           .then(data => {
               document.getElementById('age_result').textContent = data;
           });
       });
   </script>
</body>
</html>

Explanation: This HTML code defines a form to input the age. When the form is submitted, the age data is sent to the Flask route '/check_age' using a POST request. The JavaScript fetches the age data from the Flask route and updates the 'age_result' div with the response.
