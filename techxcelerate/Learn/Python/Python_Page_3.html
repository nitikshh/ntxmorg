File Handling

File handling is an essential aspect of programming that allows you to read from and write to files on the disk. Python provides built-in functions and methods for working with files.


**Opening a File in Python**

In Python, you can open files in various modes using different methods. Each method provides different capabilities and flexibility. Let's explore the different ways to open a file in Python and their respective explanations.


Method 1: Using open() function

The open() function is a built-in Python function used to open a file. It takes two parameters: the file path and the mode in which the file should be opened.


# Opening a file in read mode
file1 = open('file.txt', 'r')

# Opening a file in write mode
file2 = open('file.txt', 'w')

# Opening a file in append mode
file3 = open('file.txt', 'a')

# Opening a file in binary read mode
file4 = open('file.txt', 'rb')

# Opening a file in binary write mode
file5 = open('file.txt', 'wb')

# Opening a file in binary append mode
file6 = open('file.txt', 'ab')
Method 2: Using with statement

The with statement ensures that the file is properly closed after its suite finishes, even if an exception is raised during the operation.


# Opening a file using with statement
with open('file.txt', 'r') as file:
    # Perform operations on the file
    data = file.read()
    print(data)

Explanation: In the above examples, we demonstrate two methods to open a file in Python. The first method uses the open() function with different modes such as read ('r'), write ('w'), and append ('a'). The second method uses the with statement, which automatically closes the file after the block of code within the with statement executes.


**Reading from a File in Python**

Reading from a file in Python involves various methods to extract data from the file. You can read the entire contents of the file, read line by line, or read a specific number of characters. Let's explore the different methods for reading from a file in Python.


Method 1: Reading Entire File

You can read the entire contents of the file using the read() method.


# Open the file
with open('file.txt', 'r') as file:
   # Read entire contents of the file
   contents = file.read()
   print(contents)

   # Or using readlines() method
   # lines = file.readlines()
   # for line in lines:
   #     print(line.strip())
Method 2: Reading Line by Line

You can read the file line by line using a loop or the readline() method.


# Open the file
with open('file.txt', 'r') as file:
   # Read file line by line
   for line in file:
       print(line.strip())  # Strip newline character
       # Or using readline() method
       # line = file.readline()
       # print(line.strip())
Method 3: Reading a Specific Number of Characters

You can read a specific number of characters from the file using the read() method with a parameter specifying the number of characters to read.


# Open the file
with open('file.txt', 'r') as file:
   # Read 10 characters from the file
   data = file.read(10)
   print(data)

Explanation: In the above examples, we demonstrate different methods for reading from a file in Python. You can read the entire contents of the file using the read() method, read line by line using a loop or readline() method, or read a specific number of characters using the read() method with a parameter.


**Writing to a File**

Writing to a file in Python allows you to save data to a file. You can create a new file or overwrite an existing file. Let's explore the different methods for writing to a file in Python.


Method 1: Writing to a New File

You can write to a new file using the open() function with the write ('w') mode.


# Open a new file for writing
with open('new_file.txt', 'w') as file:
   # Write data to the file
   file.write('Hello, World!\n')
   file.write('This is a new file.\n')

Method 2: Appending to an Existing File

You can append data to an existing file using the append ('a') mode.


# Open an existing file for appending
with open('existing_file.txt', 'a') as file:
   # Append data to the file
   file.write('Additional data.\n')

Method 3: Writing to a File using writelines() Method

You can write a list of lines to a file using the writelines() method.


# Open a new file for writing
with open('new_file.txt', 'w') as file:
   # Write multiple lines to the file
   lines = ['Line 1\n', 'Line 2\n', 'Line 3\n']
   file.writelines(lines)

Explanation: In the above examples, we demonstrate different methods for writing to a file in Python. You can write to a new file using the write mode ('w'), append to an existing file using the append mode ('a'), or write a list of lines to a file using the writelines() method.


**Closing a File in Python**

Closing a file in Python is important to ensure that resources are properly released and data is saved. You can close a file using the close() method or by using the with statement.


Method 1: Closing a file using close() method

You can explicitly close a file using the close() method.


# Open the file
file = open('file.txt', 'r')

# Perform operations on the file

# Close the file
file.close()

Method 2: Closing a file using with statement

You can use the with statement, which automatically closes the file after the block of code within the with statement executes.


# Open the file using with statement
with open('file.txt', 'r') as file:
   # Perform operations on the file
   data = file.read()
   print(data)

Explanation: In the above examples, we demonstrate different methods for closing a file in Python. You can explicitly close a file using the close() method or use the with statement, which automatically closes the file after the block of code within the with statement executes.


Exception Handling in Python

Exception handling in Python allows you to handle errors and exceptions that occur during the execution of a program. This helps in gracefully handling unexpected situations and prevents the program from crashing. Let's explore how to handle exceptions in Python.


Try-Except block

You can use a try-except block to handle exceptions in Python. The code inside the try block is executed, and if an exception occurs, it is caught by the except block.


try:
   # Code that may raise an exception
   result = 10 / 0
except ZeroDivisionError:
   # Handle the exception
   print("Error: Division by zero")

Multiple Except blocks

You can use multiple except blocks to handle different types of exceptions.


try:
   # Code that may raise an exception
   result = 10 / 0
except ZeroDivisionError:
   # Handle division by zero error
   print("Error: Division by zero")
except ValueError:
   # Handle value error
   print("Error: Invalid value")

Finally block

You can use a finally block to execute cleanup code, regardless of whether an exception occurred or not.


try:
   # Code that may raise an exception
   result = 10 / 0
except ZeroDivisionError:
   # Handle division by zero error
   print("Error: Division by zero")
finally:
   # Cleanup code
   print("Cleaning up")

Explanation: In the above examples, we demonstrate how to handle exceptions in Python using try-except blocks. The try block contains the code that may raise an exception, and the except block handles the exception if it occurs. You can also use multiple except blocks to handle different types of exceptions. The finally block is executed regardless of whether an exception occurred or not, making it useful for cleanup code.

Modules and Packages

Modules and packages are essential components of Python that enable code organization, reusability, and maintainability. They allow you to break down your code into smaller, manageable units and reuse code across different projects.


**Modules**

Modules in Python are files containing Python code that can be imported and used in other Python scripts. Modules help in organizing code and making it reusable. Let's explore how to work with modules in Python.


Importing a Module

You can import a module using the import keyword followed by the module name. 


import math

# Use functions and variables from the math module
print(math.pi)
print(math.sqrt(16))

Importing Specific Functions or Variables from a Module

You can import specific functions or variables from a module using the from keyword.


from math import pi, sqrt

# Use the imported functions and variables directly
print(pi)
print(sqrt(16))

Renaming Imported Modules or Functions

You can rename imported modules or functions using the as keyword.


import math as m

# Use the renamed module
print(m.pi)

Explanation: In the above examples, we demonstrate how to work with modules in Python. You can import entire modules using the import keyword, import specific functions or variables from a module using the from keyword, or rename imported modules or functions using the as keyword. Modules help in organizing code and making it reusable.


**Creating a Module**

You can create your own modules in Python by writing Python code in a separate file and then importing and using that module in other Python scripts. Let's see how to create and use a module in Python.


Creating a Module

Create a new Python file with the desired module name and write the Python code you want to include in the module.

my_module.py:


# my_module.py

def greet(name):
   return f"Hello, {name}!"

Using the Created Module

You can import and use the functions defined in the created module in other Python scripts. 


# Import the created module
import my_module

# Use the greet function from the module
message = my_module.greet("Alice")
print(message)  # Output: Hello, Alice!

Explanation: In the above examples, we demonstrate how to create and use a module in Python. First, we create a module by writing Python code in a separate file (my_module.py). Then, we import the module using the import keyword and use the functions defined in the module in our Python script. Creating and using modules helps in organizing code and promoting reusability.


Packages in Python

Packages in Python are a way of organizing related modules into a single directory hierarchy. A package can contain subpackages, modules, or both. Let's see how to create and use a package in Python.


**Creating a Package**

To create a package, you need to organize your modules into a directory structure and include a special __init__.py file in each directory to indicate that it's a package.

Let's create a package named temperature with a module convert.py that converts temperature from Kelvin to Celsius.

temperature/convert.py:


# temperature/convert.py

def kelvin_to_celsius(kelvin):
   return kelvin - 273.15

temperature/__init__.py (empty file):


# temperature/__init__.py
# This file can be empty or contain initialization code for the package.

Using the Package

You can import the package and its modules using dot notation.


# Import the package and module
import temperature.convert as convert

# Use the kelvin_to_celsius function from the convert module
temperature_in_kelvin = 300
temperature_in_celsius = convert.kelvin_to_celsius(temperature_in_kelvin)
print(temperature_in_celsius)  # Output: 26.85
       

temperature/
├── __init__.py
├── convert.py

Explanation: In the above examples, we demonstrate how to create and use a package in Python. We organize related modules into a directory structure and include a special __init__.py file in each directory to indicate that it's a package. Then, we can import the package and its modules using dot notation and use the functions defined in the modules.

Object-Oriented Programming

Object-oriented programming (OOP) is a programming paradigm that focuses on modeling software entities as objects that have attributes (data) and methods (functions). Python is an object-oriented programming language that supports the creation and use of classes and objects.


**Classes in Python**

Classes in Python allow you to create objects that have their own attributes and methods. They provide a way to structure and organize code in a logical manner. Let's explore classes in more detail with multiple examples.


Example 1: Creating a simple class

Define a class named Person with a constructor that initializes the name attribute.


class Person:
   def __init__(self, name):
       self.name = name

# Create an instance of the Person class
person1 = Person("Alice")
print(person1.name)  # Output: Alice

Example 2: Adding methods to a class

Add a method named say_hello to the Person class.


class Person:
   def __init__(self, name):
       self.name = name
   
   def say_hello(self):
       return f"Hello, my name is {self.name}!"

# Create an instance of the Person class
person2 = Person("Bob")
print(person2.say_hello())  # Output: Hello, my name is Bob!

Example 3: Class inheritance

Create a subclass named Student that inherits from the Person class.


class Student(Person):
   def __init__(self, name, student_id):
       super().__init__(name)
       self.student_id = student_id

# Create an instance of the Student class
student1 = Student("Charlie", "S001")
print(student1.name)        # Output: Charlie
print(student1.student_id)  # Output: S001

Example 4: Class attributes

Define a class attribute named count to keep track of the number of instances created.


class Dog:
   count = 0  # Class attribute
   
   def __init__(self, name):
       self.name = name
       Dog.count += 1  # Increment count when a new instance is created

# Create instances of the Dog class
dog1 = Dog("Buddy")
dog2 = Dog("Max")
print(Dog.count)  # Output: 2

Example 5: Class methods

Define a class method named get_count to retrieve the count of instances created.


class Dog:
   count = 0  # Class attribute
   
   def __init__(self, name):
       self.name = name
       Dog.count += 1  # Increment count when a new instance is created
   
   @classmethod
   def get_count(cls):
       return cls.count

# Create instances of the Dog class
dog3 = Dog("Rocky")
dog4 = Dog("Luna")
print(Dog.get_count())  # Output: 2

Explanation: In the above examples, we demonstrate various aspects of classes in Python. We create classes with constructors, methods, and attributes. We also explore class inheritance, class attributes, and class methods. Classes provide a powerful way to model real-world entities and organize code effectively.

**Encapsulation, Inheritance, and Polymorphism in Python**

Encapsulation, inheritance, and polymorphism are fundamental concepts in object-oriented programming (OOP). Let's explore these concepts in Python with examples.


Example 1: Encapsulation

Encapsulation refers to the bundling of data (attributes) and methods that operate on the data within a single unit or class.


class Car:
   def __init__(self, brand, model):
       self.brand = brand
       self.model = model
   
   def drive(self):
       return f"Driving {self.brand} {self.model}"

# Create an object of the Car class
car1 = Car("Toyota", "Corolla")

# Access attributes and call methods
print(car1.brand)        # Output: Toyota
print(car1.model)        # Output: Corolla
print(car1.drive())      # Output: Driving Toyota Corolla

Example 2: Inheritance

Inheritance allows a class (subclass) to inherit attributes and methods from another class (superclass).


class Vehicle:
   def __init__(self, brand):
       self.brand = brand
   
   def start(self):
       return f"Starting {self.brand}"

# Car class inherits from Vehicle
class Car(Vehicle):
   def __init__(self, brand, model):
       super().__init__(brand)
       self.model = model
   
   def drive(self):
       return f"Driving {self.brand} {self.model}"

# Create an object of the Car class
car2 = Car("Ford", "Mustang")

# Access attributes and call methods
print(car2.brand)        # Output: Ford
print(car2.model)        # Output: Mustang
print(car2.start())      # Output: Starting Ford
print(car2.drive())      # Output: Driving Ford Mustang

Example 3: Polymorphism

Polymorphism allows objects of different classes to be treated as objects of a common superclass.


class Animal:
   def make_sound(self):
       pass

class Dog(Animal):
   def make_sound(self):
       return "Woof!"

class Cat(Animal):
   def make_sound(self):
       return "Meow!"

# Create objects of different classes
animal1 = Dog()
animal2 = Cat()

# Call the make_sound method
print(animal1.make_sound())  # Output: Woof!
print(animal2.make_sound())  # Output: Meow!

Explanation: In this section, we explore encapsulation, inheritance, and polymorphism in Python. Encapsulation allows data and methods to be bundled within a class. Inheritance enables classes to inherit attributes and methods from other classes. Polymorphism allows objects of different classes to be treated as objects of a common superclass, facilitating code reuse and flexibility in programming.

Regular Expressions

Regular expressions, often abbreviated as regex or regexp, are powerful tools for pattern matching and text manipulation. They provide a concise and flexible way to search, extract, and manipulate text data based on specific patterns. Python's re module provides support for working with regular expressions. Let's explore regular expressions in detail:


**Pattern Matching**

Regular expressions (regex) are powerful tools for pattern matching and text manipulation. Python provides the re module for working with regular expressions. Let's explore how to use regular expressions in Python.


Example 1: Matching a simple pattern

Check if a string contains a specific pattern using the re.search() function.


import re

# Define the pattern
pattern = r'apple'

# Search for the pattern in a string
text = 'I have an apple'
match = re.search(pattern, text)

# Check if the pattern is found
if match:
   print('Pattern found!')
else:
   print('Pattern not found.')

Example 2: Extracting matched groups

Extract specific parts of a string that match a pattern using groups.


# Define a pattern with groups
pattern = r'(\d{3})-(\d{3})-(\d{4})'

# Search for the pattern in a string
text = 'My phone number is 123-456-7890'
match = re.search(pattern, text)

# Extract matched groups
if match:
   area_code = match.group(1)
   print('Area code:', area_code)
   print('Full number:', match.group())
else:
   print('Pattern not found.')

Example 3: Substituting matched patterns

Replace matched patterns in a string with a specified value using the re.sub() function.


# Define a pattern to match
pattern = r'\bcat\b'

# Replace matched patterns in a string
text = 'I have a cat and a dog. My cat is black.'
new_text = re.sub(pattern, 'dog', text)

print(new_text)  # Output: I have a dog and a dog. My dog is black.

Example 4: Splitting a string using a pattern

Split a string into substrings using a specified pattern with the re.split() function.


# Define a pattern to split by
pattern = r'\s+'

# Split a string using the pattern
text = 'apple orange   banana'
tokens = re.split(pattern, text)

print(tokens)  # Output: ['apple', 'orange', 'banana']

Example 5: Flags in regular expressions

Use flags to modify the behavior of regular expression functions.


# Define a pattern with case-insensitive flag
pattern = r'apple'
text = 'I have an Apple'
match = re.search(pattern, text, re.IGNORECASE)

if match:
   print('Pattern found!')
else:
   print('Pattern not found.')

Explanation: Regular expressions provide a powerful way to search, extract, replace, and split strings based on patterns. In Python, the re module offers functions like search(), sub(), split(), etc., to work with regular expressions. Understanding regular expressions is essential for text processing and data manipulation tasks.


**Creating Regular Expressions in Python**

Regular expressions (regex) are patterns used to match character combinations in strings. In Python, regular expressions are created using special syntax. Let's explore how to create regular expressions in Python.


Example 1: Matching a single character

Match a single character using ordinary characters or metacharacters like ., \d, \w, etc.


import re

# Match a single digit
pattern1 = r'\d'

# Match any single character except newline
pattern2 = r'.'

# Match a word character
pattern3 = r'\w'

Example 2: Matching specific characters

Match specific characters or character ranges using square brackets [].


# Match either 'a' or 'b'
pattern1 = r'[ab]'

# Match any lowercase letter
pattern2 = r'[a-z]'

# Match any digit or whitespace character
pattern3 = r'[\d\s]'

Example 3: Anchors

Use anchors to match positions in a string: ^ for the start of the string and $ for the end of the string.


# Match a string that starts with 'hello'
pattern1 = r'^hello'

# Match a string that ends with 'world'
pattern2 = r'world$'

Example 4: Quantifiers

Use quantifiers to specify the number of occurrences of a character or group.


# Match zero or more occurrences of 'a'
pattern1 = r'a*'

# Match one or more occurrences of 'b'
pattern2 = r'b+'

# Match exactly three occurrences of 'c'
pattern3 = r'c{3}'

Example 5: Groups

Use parentheses to create groups and capture parts of a matched pattern.


# Match 'hello' followed by one or more digits
pattern1 = r'(hello)\d+'

# Capture the area code and local number from a phone number
pattern2 = r'(\d{3})-(\d{4})'

Explanation: Regular expressions are created using special syntax to define patterns for matching character combinations in strings. In Python, regex patterns are constructed using metacharacters, character classes, anchors, quantifiers, and groups. Understanding how to create regular expressions is essential for effective pattern matching and text processing tasks.


**Matching Strings**

Matching strings using regular expressions involves searching for patterns within text data. In Python, the re module provides functions to perform string matching operations. Let's explore how to match strings using regular expressions in Python.


Example 1: Using re.search()

Search for a pattern within a string using the re.search() function.


import re

# Define the pattern to search for
pattern = r'apple'

# Define the text to search within
text = 'I have an apple and a banana.'

# Search for the pattern in the text
match = re.search(pattern, text)

# Check if the pattern is found
if match:
   print('Pattern found at index', match.start())
else:
   print('Pattern not found.')

Example 2: Using re.match()

Match a pattern at the beginning of a string using the re.match() function.


# Define the pattern to match at the beginning of the string
pattern = r'apple'

# Define the text to match against
text = 'apple pie is delicious.'

# Match the pattern at the beginning of the text
match = re.match(pattern, text)

# Check if the pattern is found
if match:
   print('Pattern matched at the beginning of the text.')
else:
   print('Pattern not found.')

Example 3: Using re.findall()

Find all occurrences of a pattern within a string using the re.findall() function.


# Define the pattern to find
pattern = r'apple'

# Define the text to search within
text = 'I have an apple and a banana, but I prefer applesauce.'

# Find all occurrences of the pattern in the text
matches = re.findall(pattern, text)

# Print all matches
print(matches)

Example 4: Using re.finditer()

Find all occurrences of a pattern and their positions within a string using the re.finditer() function.


# Define the pattern to find
pattern = r'apple'

# Define the text to search within
text = 'I have an apple and a banana, but I prefer applesauce.'

# Find all occurrences of the pattern and their positions
matches = [(match.start(), match.end()) for match in re.finditer(pattern, text)]

# Print matches and their positions
for start, end in matches:
   print(f"Match found at positions {start}-{end-1}")

Example 5: Using re.fullmatch()

Match the entire string against a pattern using the re.fullmatch() function.


# Define the pattern to match the entire string
pattern = r'apple'

# Define the text to match against
text = 'apple'

# Check if the entire text matches the pattern
match = re.fullmatch(pattern, text)

if match:
   print('Text fully matches the pattern.')
else:
   print('Text does not fully match the pattern.')

Explanation: Matching strings with regular expressions involves searching for specific patterns within text data. Python's re module provides functions like search(), match(), findall(), finditer(), and fullmatch() to perform string matching operations. These functions allow you to find patterns, extract matched substrings, and obtain information about matched positions within strings.


**Character Classes**

Character classes in regular expressions allow you to match specific sets of characters. They provide a shorthand for specifying common character types or custom character ranges. Let's explore how to use character classes in regular expressions.


Example 1: Matching digits

Use \d to match any digit (0-9).


import re

# Define the pattern to match digits
pattern = r'\d'

# Define the text to search within
text = 'I have 3 apples and 5 bananas.'

# Find all digits in the text
matches = re.findall(pattern, text)

# Print all matches
print(matches)

Example 2: Matching word characters

Use \w to match any alphanumeric character (a-z, A-Z, 0-9, and underscore).


# Define the pattern to match word characters
pattern = r'\w'

# Define the text to search within
text = 'Hello World!'

# Find all word characters in the text
matches = re.findall(pattern, text)

# Print all matches
print(matches)

# Define the pattern to match whitespace characters
pattern = r'\s'

# Define the text to search within
text = 'Hello\tWorld\n'

# Find all whitespace characters in the text
matches = re.findall(pattern, text)

# Print all matches
print(matches)

Example 4: Negating character classes

Use \D, \W, and \S to match any character except digits, word characters, and whitespace characters, respectively.


# Define the pattern to match non-digits

Explanation: Character classes in regular expressions provide a concise way to match specific sets of characters. Python's re module allows you to use shorthand notations like \d, \w, \s to match digits, word characters, and whitespace characters, respectively. Additionally, you can define custom character classes using square brackets to match specific character sets or ranges.


**Anchors and Quantifiers in Regular Expressions**

Anchors and quantifiers are essential components of regular expressions used to specify positions and repetition in patterns. Anchors define where a pattern should occur within a string, while quantifiers specify how many times a character or group should repeat. Let's explore how to use anchors and quantifiers in regular expressions.


Example 1: Start of string anchor (^)

Use ^ to match the start of a string.


import re

# Define the pattern to match
pattern = r'^Hello'

# Define the text to search within
text = 'Hello World!'

# Check if the pattern matches the start of the text
if re.search(pattern, text):
   print('Pattern found at the start of the text.')
else:
   print('Pattern not found at the start of the text.')

Example 2: End of string anchor ($)

Use $ to match the end of a string.


# Define the pattern to match
pattern = r'World$'

# Define the text to search within
text = 'Hello World!'

# Check if the pattern matches the end of the text
if re.search(pattern, text):
   print('Pattern found at the end of the text.')
else:
   print('Pattern not found at the end of the text.')

Example 3: Zero or one quantifier (?)

Use ? to match zero or one occurrence of a character or group.


# Define the pattern to match
pattern = r'colou?r'

# Define the text to search within
text1 = 'color'
text2 = 'colour'

# Check if the pattern matches the text
if re.search(pattern, text1):
   print('Pattern found in text1.')
if re.search(pattern, text2):
   print('Pattern found in text2.')

Example 4: Zero or more quantifier (*)

Use * to match zero or more occurrences of a character or group.


# Define the pattern to match
pattern = r'go*d'

# Define the text to search within
text1 = 'god'
text2 = 'good'
text3 = 'gooooood'

# Check if the pattern matches the text
if re.search(pattern, text1):
   print('Pattern found in text1.')
if re.search(pattern, text2):
   print('Pattern found in text2.')
if re.search(pattern, text3):
   print('Pattern found in text3.')

Example 5: One or more quantifier (+)

Use + to match one or more occurrences of a character or group.


# Define the pattern to match
pattern = r'go+d'

# Define the text to search within
text1 = 'god'
text2 = 'good'
text3 = 'gooooood'

# Check if the pattern matches the text
if re.search(pattern, text1):
   print('Pattern found in text1.')
if re.search(pattern, text2):
   print('Pattern found in text2.')
if re.search(pattern, text3):
   print('Pattern found in text3.')

Explanation: Anchors (^ and $) specify positions within a string where patterns should occur. Quantifiers (?, *, and +) define the number of occurrences of characters or groups. Using anchors and quantifiers in regular expressions allows for precise pattern matching and text processing tasks.
